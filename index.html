retail-weather-pipeline
GitHub ↗ Dashboard ↗

An end-to-end ELT pipeline for a hypothetical multi-city retailer needing to understand how weather drives inventory demand variability across markets. Pulls daily weather data for 5 US cities via the Open-Meteo API, lands raw JSON in GCS, loads to BigQuery, transforms through dbt, orchestrates with Dagster, and surfaces insights in Looker Studio.
